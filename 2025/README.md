# Advent of Code 2025 - Rust

Solutions for Advent of Code 2025 implemented in Rust.

## Structure

Each day is organized as a separate Cargo project:

```
2025/
├── day1/
│   ├── Cargo.toml
│   ├── src/
│   │   └── main.rs
│   └── input.txt
├── day2/
│   ├── Cargo.toml
│   ├── src/
│   │   └── main.rs
│   └── input.txt
└── ...
```

## Running Solutions

To run a specific day's solution:

```bash
cd dayX
cargo run [part1|part2]
```
